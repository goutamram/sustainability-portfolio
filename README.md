## 📁 Portfolio Projects

### 🔬 [Project 1: LCA Calculator - Coffee Supply Chain](./01_LCA_Coffee_Analysis)
**Python-based LCA model for coffee production from farm to cup**
- Quantifies environmental impacts across lifecycle stages
- Compares conventional vs. sustainable farming methods
- Generates impact reports (GWP, AP, EP, WD)
- **Tech Stack:** Python, Pandas, Matplotlib, LCA methodology

### 🌱 [Project 2: Carbon Footprint Dashboard](./02_Carbon_Footprint_Dashboard)
**Interactive Power BI dashboard for organizational carbon tracking**
- Scope 1, 2, 3 emissions tracking and analysis
- Year-over-year emissions trends
- Departmental breakdown and reduction targets
- Data visualizations for stakeholder reporting
- **Tech Stack:** Power BI, Excel, DAX, Carbon Accounting

### 🗺️ [Project 3: Geospatial Environmental Impact Mapping](./03_Geospatial_Impact_Analysis)
**GIS-based analysis of industrial project environmental footprint**
- Land use change assessment
- Biodiversity impact mapping
- Water resource footprint analysis
- Stakeholder visualization tools
- **Tech Stack:** QGIS, Python, Geopandas, Remote Sensing Data

### ♻️ [Project 4: Circular Economy Waste Flow Analysis](./04_Circular_Economy_Analysis)
**Material flow analysis and circular economy assessment**
- Waste stream quantification and categorization
- Material recycling potential calculation
- Circular economy index development
- **Tech Stack:** Python, Sankey diagrams, Material Flow Analysis
